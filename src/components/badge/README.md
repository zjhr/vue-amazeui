>徽章

#prop

##amStyle
>type : String
>>颜色，参数【blue，sblue，green，orange，red，black】

##blue
>type : Boolean
>>蓝色

##sblue
>type : Boolean
>>浅蓝色

##green
>type : Boolean
>>绿色

##orange
>type : Boolean
>>橙色

##red
>type : Boolean
>>红色

##black
>type : Boolean
>>黑色

##rounded
>type : Boolean
>>渲染为圆角微章

##elm
>type : String
>>渲染标签元素，默认span，参数为正确的标签名

##href
>type : String
>>渲染为a标签

##target
>type : Boolean,String
>>渲染为a标签时候的打开链接类型
