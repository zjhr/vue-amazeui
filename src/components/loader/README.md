>type : Boolean

##green
>type : Boolean
>>绿色

##orange
>type : Boolean
>>橙色

##red
>type : Boolean
>>红色

##black
>type : Boolean
>>黑色

##white
>type : Boolean
>>白色

##rounded
>type : Boolean
>>渲染为圆角
