# offcanvas
---
>侧边栏组件

#prop

##open
>type : Boolean
>>是否打开

##silde
>type : String
>>滑出方向 默认为左侧滑出，left,right,top,bottom

##sildewh
>type : String
>>offcanvas的宽度，left,right为宽度默认70%,top,bottom为相对末端的距离默认70%

#emit

##Close
>点击遮罩时候的回调

##onOpen
>侧边栏打开之前的回调

#onClosed
>侧边栏关闭之后的回调
