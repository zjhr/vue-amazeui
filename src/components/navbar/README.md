# navbar
---
>导航条组件

#prop

##amStyle
>type : String
>>颜色，参数【blue，sblue，green，orange，red，black】

##blue
>type : Boolean
>>蓝色

##sblue
>type : Boolean
>>浅蓝色

##green
>type : Boolean
>>绿色

##orange
>type : Boolean
>>橙色

##red
>type : Boolean
>>红色

##black
>type : Boolean
>>黑色

##titleOnLeft
>type : Boolean
>>主标题是否居左，默认居中。
